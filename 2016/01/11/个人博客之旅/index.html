个人博客之旅（一）— 博客搭建

准备工作：

设备：Mac

博客驱动：Hexo

部署平台：GitHub

需要安装：Git、Node.js、npm

一、安装Git

下载安装程序：Git下载地址 or Git官网

二、安装Node.js

地址：Node.js官网

网站内有两个版本下载，这里选择第一个LTS版本。安装过程中，Node.js和npm会一并安装。可在终端输入node -v检查是否安装成功(安装成功会有相关介绍打印)

三、安装Hexo

完成上面两步后，就可安装Hexo了。
不能翻墙的，可以换个源安装

npm install hexo --no-optional

终端执行（安装Hexo需要电脑安装了Xcode）

sudo npm install hexo -g

安装完之后，执行 hexo -v 查看是否安装成功

安装完成之后，在任意文件夹内创建一个新的文件夹，这里以桌面为例创建一个
/Desktop/MyBlog/Hexo

终端一次执行

cd Desktop/MyBlog/Hexo
hexo init

成功时打印

继续执行

npm install
hexo server


在浏览器中打开地址：http://0.0.0:4000，显示如下界面即成功。

四、GitHub部署

在GitHub上创建账号，完成后点击页面右上角“+” - New repository创建新仓库。

填写仓库名，格式必须为 [账户名].github.io，我这里因为已经创建过，所以提示红 ! 号，正常情况为绿色 √ 号.

创建完成后，在Desktop/MyBlog/Hexo目录下使用编辑器打开_config.yml文件，在最下面添加：repository为你刚刚在GitHub上创建的仓库的地址

在终端执行（此时终端应当还在Hexo文件夹下）开始部署

hexo deploy

部署完成后，在浏览器中直接打开地址 http://mokyz.github.io 即可打开刚刚搭建的博客

五、发表博客

终端执行

hexo new "博客的标题"

创建完之后，会生成MarkDown文件 hexo/source/_posts/博客的标题.md，可以使用MarkDown编辑器进行编辑。编辑完之后，在终端执行

hexo g    // 生成静态网页
hexo s    // 本地预览，可在浏览器打开 0.0.0.0:4000 预览效果
hexo d    //    部署到github
